# S520-B 代码分析报告

## 1. 项目概述

| 项目 | 说明 |
|------|------|
| 产品名称 | DJM-S520 机械理疗仪（S520-B 型号） |
| 系统组成 | 主控板（STM32F103）+ Bootloader + 机械手柄（STM8S） |
| 开发环境 | Keil MDK-ARM（主控/Bootloader）、IAR STM8（手柄） |
| 固件库 | STM32F10x 标准外设库 V3.5.0、STM8S 标准库 |

### 子系统关系

- **主控板**：系统核心，负责蓝牙通信、APP指令解析、手柄控制中转、压力PID调节、压缩机/电磁阀控制、LED/蜂鸣器状态指示
- **Bootloader**：上电时优先运行，检查升级标志，决定跳转APP或进入固件升级模式
- **机械手柄**：接收主控板指令，通过 PWM 驱动直流电机实现推杆前进/后退，带限位保护和 EEPROM 位置记忆

**关键函数：**

| 函数 | 说明 |

**数据流：**
```
蓝牙模块 → USART2 RX DMA → 环形缓冲区 → 解码状态机 → 命令分发
  ├─ 开关机 → Flag.WorkStatus
  ├─ 机械手控制 → USART4 转发到手柄
  └─ 升级指令 → Flag.update
```

#### 3.1.7 压力PID模块 (Pressure_PID.c / Pressure_PID.h)

| 项目 | 说明 |
|------|------|
| 功能 | 气压PID闭环控制算法 |
| 算法 | 支持位置式PID和增量式PID（编译开关选择） |
| PID参数 | Kp=0.01, Ki=0.001, Kd=0.1 |
| 目标压力 | 3200（对应约320Kpa） |
| 偏差门限 | 500（超过此值取消积分项，仅用PD） |
| 输出范围 | 1000~4000（对应100Kpa~400Kpa） |
| 调试 | 支持 VOFA+ WaterFire 协议输出 |

**位置式PID公式：**
```
u(k) = Kp * err(k) + Ki * ∑err(k) + Kd * [err(k) - err(k-1)]
```

**增量式PID公式：**
```
Δu(k) = Kp * [err(k) - err(k-1)] + Ki * err(k) + Kd * [err(k) - 2err(k-1) + err(k-2)]
u(k) = u(k-1) + Δu(k)
```

**控制逻辑：**
- PID计算值 > 3400 时，关闭二号电磁阀（泄压）
- 否则开启二号电磁阀（保压）

### 3.2 IAP Bootloader（STM32F10x）

#### 3.2.1 系统初始化

| 项目 | 说明 |
|------|------|
| 时钟 | HSE 8MHz → PLL × 9 = 72MHz |
| 中断向量表 | FLASH_BASE (0x08000000) |
| 串口 | USART2, 9600bps（仅用于固件升级） |
| 其他串口 | USART1/3/UART4 在 `peripheral_init()` 中禁用 |

#### 3.2.2 升级状态机 (Iap.c / Iap.h)

**Flash 地址分配：**

| 地址 | 用途 |
|------|------|
| 0x08000000 | Flash 基地址 |
| 0x08004000 | APP 起始地址 (`IAP_ADDR`) |
| 0x0801FFF0 | 升级标志位 (`FLASH_SAVE_ADDR`) |
| 末尾1页 | 参数存储 (`IAP_PARAM_ADDR`) |

**升级状态枚举：**

| 状态 | 说明 |
|------|------|
| `eStart` | 开始，发送 ACK |
| `eCmdID` | 等待接收设备命令（MAJOR/MINOR） |
| `eDeviceVerify` | 设备验证 |
| `eMajorUpdateGrade` | 主板固件升级 |
| `eMinorUpdateGrade` | 子板固件升级 |

**升级流程：**
1. Bootloader 上电，读取升级标志
2. 若标志为1，进入升级模式
3. 发送 "UPGR" 提醒上位机
4. 接收 "MAJOR" 或 "MINOR" 命令
5. 调用 `DownloadFirmware()` 执行 Ymodem 接收
6. 写入 Flash
7. 清除升级标志，系统复位

#### 3.2.3 Flash 操作 (Iap.c)

**关键函数：**

| 函数 | 说明 |
|------|------|
| `IAP_UpdateProgram()` | 按字(32位)编程，带验证和重试 |
| `IAP_FlashEease()` | 按页擦除指定大小 |
| `IAP_DisableFlashWPR()` | 关闭写保护 |
| `IAP_UpdateParam()` | 更新参数到 Flash |
| `IAP_JumpToApplication()` | 跳转到 APP (0x08004000) |

**编程策略：**
- 按 4 字节(32位)对齐写入
- 每次写入后立即验证

## 4. 通信协议分析

### 4.1 主控 ↔ 手柄（UART4）

- **波特率**: 9600bps
- **帧格式**: 0x55AA + Len + DeviceID(0x0001) + CmdType + CmdStatus + Data + CRC16
- **CRC算法**: CRC16-Modbus (多项式 0xA001)
- **设备ID**: 0x0001

### 4.2 主控 ↔ 蓝牙/APP（USART2）

- **波特率**: 9600bps
- **帧格式**: 与手柄相同
- **命令类型**: ACT(0x31) / SET(0x32) / GET(0x33)

### 4.3 Bootloader ↔ 上位机（USART2）

- **波特率**: 9600bps
- **协议**: Ymodem（128/1024字节数据包 + CRC16-CCITT）
